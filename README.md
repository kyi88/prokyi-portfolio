# My Site

シンプルでエレガントなWebサイト

## 機能

- レスポンシブデザイン
- モダンなUI/UX
- アニメーション効果

## ファイル構成

```
my-site/
├── index.html     - メインページ
├── style.css      - スタイル
├── script.js      - JavaScript
├── .gitignore     - Git設定
└── README.md      - このファイル
```

## ローカルで実行

```bash
# シンプルなサーバーで実行（Python）
python3 -m http.server 8000

# または Node.js が必要な場合
npx http-server
```

ブラウザで `http://localhost:8000` にアクセス
