## 📋 プロジェクト概要

このポートフォリオサイトは、2000年代の懐かしいWebデザインをテーマに、ぷろきぃのプロフィール、学習経歴、技術スキルを紹介するサイトです。

**特徴:**
- 🎨 2000年代レトログ風デザイン
- 📊 アクセスカウンター機能
- 🎉 キリ番（ぞろ目）表示機能
- 📱 レスポンシブ対応
- 🌐 GitHub Pages で公開

## 🔢 アクセスカウンター

ページにアクセスするたびにカウンターが増加します。

**キリ番候補:**
- `111`, `222`, `333`, `444`, etc.
- `100`, `1000`, `10000`, etc.
- ぞろ目全般

キリ番到達時は画面中央にポップアップが表示されます！ 🎉
