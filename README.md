# ★★★ prokyiの部屋 ★★★

2000年代初期の懐かしい雰囲気を完璧に再現した平成レトロ風ポートフォリオサイト。

## 特徴

- **テーブルレイアウト**: 昔ながらのHTML4.01スタイル
- **レトロなCSSスタイリング**: 古いプロパティと3D効果（insetボーダー、ドロップシャドウなど）
- **SVG埋め込みアイコン＆アニメーション**: バナー、アバター、工事中gif など
- **アクセスカウンター**: ブラウザローカル保存（擬似版、BASE=77000から開始）
- **キリ番機能**: ぞろ目検出でポップアップ表示
- **古いWeb的相互作用**: リンククリック時のボタン押下感、カウンターマウスオーバー効果
- **マーキー動画テキスト**: ヘッダーにスクロールテキスト
- **点滅表現**: `<blink>`アニメーション相当の CSS animation

## 使い方

### ローカルサーバ起動

```bash
cd ~/prokyi-portfolio
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス。

### カウンターについて

- アクセスするたびに **ブラウザのローカルストレージ** に数を記録
- 初期値: BASE=77000
- 同じブラウザで何度訪問してもカウントが増える（デモ用）
- **実装例**: サーバ側DB連携で本物のカウンターにできます

### カスタマイズ

- **セクション追加**: HTMLの `<h2>` ごとに新規セクションを追加
- **リンク集**: `#links` セクション内の `<a>` タグを編集
- **色調変更**: `style.css` の色コード（例 `#ffd`, `#003366`）を編集
- **アバター差し替え**: SVG コード（`<svg class="avatar">` 内）を編集
